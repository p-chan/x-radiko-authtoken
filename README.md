# x-radiko-authtoken

> A Library to get a valid x-radiko-authtoken

## Install

```shell
$ npm install x-radiko-authtoken
```

## Usage

```js
import { getAuthToken } from 'x-radiko-authtoken'

getAuthToken().then((authToken) => {
  console.log(authToken)
})
```
